Biến khối SketchUp thành ảnh diễn họa chân thực bằng AI · Giữ nguyên hình học

✓ Yêu cầu trước khi cài đặt
🐍
Python 3.10 trở lên Tải tại python.org/downloads — khi cài nhớ tích ô "Add Python to PATH" ở màn hình đầu tiên.
🌐
Kết nối Internet Cần để gọi API Google Gemini mỗi lần render. Không cần GPU.
🔑
Gemini API Key (Google AI Studio) Truy cập aistudio.google.com/apikey → tạo key miễn phí. Lưu ý: tính năng tạo ảnh yêu cầu bật Billing trên Google Cloud (khoảng $0.04–0.15 / ảnh).
⚠️ Quan trọng khi cài Python: Nếu bỏ qua ô "Add Python to PATH", app sẽ không chạy được. Gỡ cài đặt và cài lại nếu cần.
1 Giải nén và chuẩn bị
1
Giải nén file ZIP

Click chuột phải vào HungArch_AI_Render_v1.0.0.zip → Extract All → chọn thư mục bất kỳ (ví dụ: D:\HungArch).

2
Mở thư mục vừa giải nén

Bên trong bạn thấy các file: run.bat, backend/, frontend/, README.md...

💡 Không cài đặt thêm bất cứ thứ gì — run.bat tự động cài tất cả thư viện cần thiết trong lần chạy đầu tiên.
2 Chạy ứng dụng
1
Double-click vào file run.bat

Một cửa sổ đen (Command Prompt) hiện lên — đây là bình thường, đừng đóng cửa sổ này trong khi dùng app.

2
Chờ lần đầu cài thư viện

Lần đầu chạy mất 2–5 phút để tải thư viện. Các lần sau chỉ mất vài giây. Trình duyệt tự mở khi sẵn sàng.

3
Trình duyệt mở http://localhost:8000

Nếu trình duyệt không tự mở, gõ địa chỉ trên vào thanh địa chỉ của Chrome / Edge.

Muốn tắt app: Đóng cửa sổ đen (Command Prompt) hoặc nhấn Ctrl + C bên trong đó. Đóng trình duyệt không tắt server.
3 Nhập Gemini API Key (lần đầu)
1
Truy cập aistudio.google.com/apikey

Đăng nhập bằng tài khoản Google → bấm "Create API Key" → copy key (dạng AIza...).

2
Dán key vào app

Trong app, bấm vào mục 🔑 Cấu hình khóa API (góc trên cùng bên trái) → dán key vào ô → bấm Lưu.

3
Góc trên phải hiện ● API sẵn sàng

Key được lưu trên máy bạn, không truyền đi đâu khác. Chỉ cần nhập 1 lần.

4 Dùng thử & lấy Mã Máy để kích hoạt bản quyền
1
App tự tạo 30 ngày dùng thử

Một banner vàng xuất hiện ở đầu trang: "Đang dùng thử — còn X ngày." Bạn dùng đầy đủ tính năng trong suốt thời gian này.

2
Lấy Mã Máy để mua bản quyền

Bấm nút "Kích hoạt bản quyền" trên banner vàng → cửa sổ hiện Mã Máy của bạn:

Mã máy của bạn

F646 – 8535

(ví dụ minh hoạ — mã thật sẽ khác)

Chụp màn hình hoặc ghi lại mã này, nhắn cho HungArch qua Zalo để nhận key.

3
Nhận License Key và kích hoạt

Sau khi nhận key từ HungArch, dán vào ô "Nhập license key" trong cùng cửa sổ → bấm ✅ Kích hoạt → dùng không giới hạn thời gian.

✅ Key được ràng buộc với máy tính của bạn — không cần Internet để xác thực. Nếu thay máy hoặc cài lại Windows, liên hệ HungArch để được cấp key mới miễn phí.

Cửa sổ đen hiện rồi đóng ngay

Python chưa được thêm vào PATH. Gỡ Python, cài lại, nhớ tích ô "Add Python to PATH".

!
Trình duyệt hiện "localhost refused to connect"

Server chưa khởi động xong — chờ thêm 30 giây rồi tải lại trang. Hoặc kiểm tra cửa sổ đen có thông báo lỗi không.

!
Render báo lỗi "API key không hợp lệ" hoặc "429"

Kiểm tra API key đã nhập đúng chưa. Nếu gặp lỗi 429: tài khoản Google Cloud chưa bật Billing hoặc đã hết quota.
